Différents dossiers présents :

- "Programme MC" : Contient le script .c de la simulation et le fichier de plot. 
1) Compiler "MC.c" avec g++
2) Exécuter le fichier "a.out"
3) Lire le dossier "plot.gnu" depuis gnuplot
Remarque : Si L est modifié, penser à reporter ce changement dans le fichier de plot aux lignes 9 et 10 pour avoir une animation centrée.

- "Résultats" : Contient trois dossiers nommés "Température", "Isotherme" et "Transition de phase"
Chacun de ces dossiers contient les figures présentées dans le compte-rendu ainsi que les fichiers de données utilisés pour les obtenir. Sont aussi présents les fichiers de plot correspondants. Les paramètres utilisés sont rappelés en commentaires.

- "Vérifications préliminaires" : Contient les figures obtenues dans la partie "Tests de vérification" du compte-rendu. Les paramètres utilisés pour les obtenirs sont rappelés dans "params.txt"
